# AfriTalent
Projet fil rouge — Plateforme de mise en relation entre freelances africains et
clients.
Auteur : MAMADOU NIANG
Promotion : L1 Web — ISI


## 📋 Description

**AfriTalent** est une plateforme web dynamique qui connecte les freelances africains talentueux avec des clients du monde entier. Le site présente une interface moderne, responsive et interactive avec des animations au scroll, un filtrage dynamique et une validation de formulaire avancée.

### 🎯 Objectif du projet

- Mettre en valeur les compétences des freelances africains
- Offrir une interface utilisateur fluide et agréable
- Démontrer des compétences en développement front-end
- Créer une vitrine professionnelle pour les talents


### les_pages_du_sites

 **Accueil (`index.html`)** | Présentation de l'entreprise, services, statistiques animées, témoignages |
 **Freelances (`freelances.html`)** | Liste des freelances avec filtrage par catégorie et recherche |
 **Tarifs (`tarifs.html`)** | Grille des offres et tarifs |
 **À propos (`about.html`)** | Histoire, équipe, valeurs et chiffres clés |
 **Contact (`contact.html`)** | Formulaire de contact avec validation JavaScript |

 ### ✨ Fonctionnalités interactives

- ***Mode Clair/Sombre** : Bascule automatique avec mémorisation
- ***Compteurs animés** : Les statistiques s'animent au scroll (Commit 7)
- ***Animations fade-in** : Apparition fluide des sections (Commit 7)
- ***Filtrage dynamique** : Filtrage des freelances sans rechargement (Commit 8)
- *** Validation de formulaire** : Validation avancée avec messages d'erreur (Commit 8)
- ***Bouton retour en haut** : Navigation rapide
- *** Design responsive** : Compatible mobile, tablette et desktop
